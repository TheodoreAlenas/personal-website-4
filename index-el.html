Θοδωρής

Φοιτητής πληροφορικής με κλίση στα Linux και στα πρωτότυπα.

Για πτυχίο θα χρειαστώ πρακτική άσκηση, το καλοκαίρι του 2025 ή λίγο αργότερα. Λέγομαι Θοδωρής Δημακόπουλος.

CV PDF
άντρας με κουκούλα, μαλλί κάτω από τον όμο, ελάχιστο μούσι και πράσινα μάτια

Εμπειρία

  • Φοιτώ στο ΕΚΠΑ από το 2019 και μάλλον μέχρι το Φεβρουάριο του 2026. Έχω συνεργαστεί με 5-10 ομάδες. Έχω κλίση σε μαθήματα με C/C++.
  • Προγραμμάτιζα πατέντες για χόμπυ και τις χρησιμοποιούσα για λίγα χρόνια. Περνούσαν από φάσεις ώστε να χρειάζονται λιγότερα εργαλεία.
  • Το 2024 έκανα πρακτική άσκηση στη Helvia. Έγραφα Docker files για να συνδέσω δύο repos και μετά έφτιαξα μια web εφαρμογή για άσκηση.

Χαρακτήρας

  • Ψάχνω τους λόγους πίσω από τον τρόπο που λειτουργεί μια βιβλιοθήκη, που συχνά λύνει απορίες στην ομάδα.
  • Είμαι πάντα προετοιμασμένος να εκλαϊκεύσω όσα καταλαβαίνω από πληροφορική.
  • Πετάω τον κώδικά μου άφοβα. Ειδικά όταν δε γνωρίζει η ομάδα ότι ασχολήθηκα με ένα κομμάτι κώδικα, είμαι έτοιμος να το πετάξω όλο. Έχω κλίση και σε user tests που ακυρώνουν UI designs μου.

Τεχνικές δεξιότητες

  • Χρησιμοποιώ Linux απ' το 2022 και είμαι εξοικειωμένος με το σύστημα αρχείων και με scripting. Από Docker ξέρω λιγότερα.
  • Έχω κάποια επαφή με OOP, design patterns και C++, Java, Python. Παλιότερα είχα εμμονή με άτομα σαν το Robert C. Martin.
  • Πειραματιζόμουν πολύ με CSS και μπορούσα να σχεδιάζω διακοπτάκια και διαχωριστικές γραμμές. Από βιβλιοθήκες ξέρω λιγότερα.

Μηνύματα ανά γράμμα

Μέρος της πρακτικής άσκησης ήταν να εξασκηθώ στις διαδυκτιακές εφαρμογές, με ReactJS, NextJS, CSS, JavaScript και Python με τη βιβλιοθήκη FastAPI. Η ιδέα πίσω από αυτή την εφαρμογή είναι δύο άτομα να στέλνουν γραπτά μηνύματα αλλά και να μπορεί ο ένας να διακόψει τον άλλο. Χρειάστηκαν αρκετοί αυτοσχεδιασμοί.

Να αναφέρω ότι έχω κάποια κουσούρια γιατί μέχρι το 2023 ήμουν πολύ απλός, σα στερεοτυπικός προγραμματιστής αλλά με πολύ έντονα αρνητικά. Σήμερα έχω δυσκολίες σε ανταγωνιστικά περιβάλλοντα γιατί τα βλέπω σαν τον τότε εαυτό μου. Συνεργάζομαι, αλλά χάνω την όρεξή μου όταν ο ένας δεν αναρωτιέται για τα νέα του άλλου ή όταν ο ένας εξηγεί στον άλλο μια ιδέα λέγοντάς του τι να πατήσει στο λάπτοπ. Θα παραμείνω στην πληροφορική για 5-10 χρόνια, για να βελτιωθώ και να κάνω καλή δουλειά, αλλά δεν είναι το όνειρό μου να μείνω για πάντα. Θέλω να πιστεύω ότι δε φέρνω αρνητική ενέργεια στην ομάδα, δεν τσακώνομαι, αλλά μπορεί μια συνεργασία να γίνει άβολη και να με αφήσουν πίσω χωρίς να το σκεφτούν. Τελευταία υποκύπτω όλο και περισσότερο στις προτιμήσεις των άλλων και στόχος μου είναι να παλεύω τις πρώτες μία ή δύο εβδομάδες για κάποιο status, γιατί συνήθως σχηματίζεται μια ιεραρχία εντός της ομάδας, και μετά θα ακολουθώ όποιον ακολουθούν οι περισσότεροι. Συμφωνώ ότι η "προγραμματιστική κουλτούρα" είναι λειτουργική και η κοινότητα είναι ενδιαφέρουσα με ποικιλία και πολλά για να εξερευνήσει κανείς.
