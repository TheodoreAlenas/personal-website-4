  6. Προτιμήσεις

    Ας είμαι εσωστρεφής, προτιμώ ένα κοινωνικό περιβάλλον που να μιλάμε για εξόδους, να κοιτάμε ο ένας την οθόνη του άλλου, να κοροϊδεύουμε, να υπάρχουν άτομα με ενέργεια, παρόλο που δε θα ταιριάζω πολύ μαζί τους. Δεν έχω θέμα με εσωστρεφείς ομάδες, αλλά είχα δυσάρεστες εμπειρίες. Προτιμώ να είμαι με εξωστρεφή άτομα που γελάνε μαζί μου παρά να πρέπει να συννενοηθούμε για κάτι και να θέλουμε να το αναβάλλουμε. Σε δύο εργασίες με διαφορετικές ομάδες, ήμουν ο μόνος που δεν πέρασε το μάθημα. Με δύο άλλες ομάδες με απέφευγαν και την τελευταία μέρα της προθεσμίας ζητούσαν να τους διορθώσω προβλήματα. Σε πολλές περιπτώσεις πρότεινα τρόπους να δουλέψουμε πιο γρήγορα και πιο ξεκούραστα και κράταγαν κόντρα πολύ δυνατά χωρίς να εξηγούν γιατί. Συνήθως, με πιο κοινωνικές ομάδες τα πάμε καλύτερα και επειδή είναι πιο ευχάριστη η συνεργασία είμαι και πιο αποδοτικός. Μπορώ να προσαρμοστώ βέβαια αν χρειάζεται.

    Η ιδανική ομάδα για μένα θα ήταν μια που ξέρει να ακολουθεί κάποια μεθοδολογία ώστε να δοκιμάζει κανείς ένα μικρό κομμάτι κώδικα εύκολα και γρήγορα, π.χ. να γράφουμε γρήγορα αυτόματα τεστ παράλληλα με το υπόλοιπο πρόγραμμα. Επειδή σε κάποιες εργασίες είχα γράψει τον κώδικα ώστε να μπορώ να τον δοκιμάζω συχνά όσο τον γράφω, ίσως έχω κάποιες από τις δεξιότητες που θα χρειαζόμουν για να συμμετέχω στην ομάδα. Η εφαρμογή με τα μηνύματα ανά γράμμα πιο πάνω έχει αρκετά τεστ που τρέχουν αρκετά γρήγορα ώστε να τα τρέχω κάθε φορά που αλλάζω μια γραμμή κώδικα. Κάποια τεστ παίρνουν μερικά δευτερόλεπτα επειδή συνδέουν το backend και μέρος του front end, αλλά είναι γρήγορα αυτά που δοκιμάζουν τη σελιδοποίηση των δεδομένων, τη μετατροπή της εισόδου σε ένα σχήμα που θα μετατραπεί σε UI με λίγο κώδικα και δοκιμάζουν τι θα συνέβαινε αν κάποιο δεδομένο είναι κενό. Νομίζω καταλαβαίνω πώς λειτουργούν αυτά.

    Τέλος, προτιμώ τη συνεργασία από κοντά. Πράγματι από απόσταση είναι καλύτερα κάποιες φορές αλλά στην πρακτική μου στη Helvia, ήταν πολύ πιο εύκολο να κάνω μια ερώτηση από κοντά στο γραφείο ή έστω να πω κάτι ενδιαφέρον που είδα. Από κοντά φαίνεται ποιος είναι συγκεντρωμένος, ποιος κοιτάει προς τα πέρα κουρασμένος, ποιος σκέφτεται κάτι ευχάριστο και ποιος έχει όρεξη να μιλήσει γενικά. Επίσης, είναι εύκολο να ρίξεις μια ματιά στην οθόνη του άλλου. Αυτό είναι και θετικό και αρνητικό αλλά έχω ανοίξει ωραίες συζητήσεις ρίχνοντας μια ματιά τι κάνει ο άλλος. Αλλά το κυριότερο είναι ότι αν κάνεις μια ερώτηση και ο άλλος θέλει να δώσει μια μεγάλη απάντηση, έχεις μια ευκαιρία να πετάξεις μια κουβέντα στη μέση για να αλλάξεις την πορεία της συζήτησης. Μπορώ να συνεργαστώ και από μηνύματα, αλλά προτιμώ τα προβλήματα της συνεργασίας από κοντά παρά τα προβλήματα της απόστασης.
